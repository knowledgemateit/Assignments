## **Part 1: Process Fundamentals**

### **Understanding Processes**

```bash
#!/bin/bash

# Display current process
ps

# Display all processes
ps aux

# Display process tree
pstree

# Get process ID
echo "Current PID: $$"
echo "Parent PID: $PPID"

# List running jobs
jobs
jobs -l  # With PID
```

### **Process Status**

```bash
#!/bin/bash

# Show detailed process info
ps -ef | grep bash

# Monitor processes in real-time
top

# Interactive process viewer
htop

# Show process hierarchy
ps -H

# Display process state
ps aux | head -1
```

## **Part 2: Background and Foreground Jobs**

### **Running Background Jobs**

```bash
#!/bin/bash

# Run command in background
command &

# Example: long-running task
sleep 100 &
echo "PID: $!"

# Multiple background jobs
./script1.sh &
./script2.sh &
./script3.sh &

# Wait for all background jobs
wait
```

### **Job Control**

```bash
#!/bin/bash

# Suspend current job (Ctrl+Z)
# Then use job control commands

# Resume job in foreground
fg %1

# Resume job in background
bg %1

# List jobs
jobs

# Bring specific job to foreground
fg %2

# Send job to background
# (suspend with Ctrl+Z, then use bg)
```

## **Part 3: Signal Handling**

### **Process Signals**

```bash
#!/bin/bash

# Common signals
# SIGTERM (15) - Termination signal
# SIGKILL (9) - Kill signal (cannot be caught)
# SIGSTOP (19) - Stop signal
# SIGCONT (18) - Continue signal
# SIGHUP (1) - Hangup
# SIGINT (2) - Interrupt (Ctrl+C)

# Send signal to process
kill -SIGTERM 1234
kill -15 1234
kill -9 1234  # Force kill

# Kill all processes matching pattern
killall bash
pkill -f "pattern"
```

## **Part 4: Wait and Process Synchronization**

### **Wait Command**

```bash
#!/bin/bash

# Wait for background job
long_task &
PID=$!
wait $PID
echo "Task completed"

# Wait for all background jobs
background_job1 &
background_job2 &
background_job3 &
wait
echo "All jobs done"

# Wait with timeout
timeout 30 long_command
```
