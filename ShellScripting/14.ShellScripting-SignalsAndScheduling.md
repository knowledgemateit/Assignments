## **Part 1: Signal Handling with Trap**

### **Trap Basics**

```bash
#!/bin/bash

# Trap SIGINT (Ctrl+C)
trap 'echo "Caught interrupt"; exit 0' INT

# Trap multiple signals
trap 'cleanup' EXIT INT TERM

cleanup() {
    echo "Cleaning up..."
    rm -f /tmp/tempfile
}

# Main logic
while true; do
    echo "Running... (Ctrl+C to stop)"
    sleep 1
done
```

## **Part 2: Cron Jobs**

### **Cron Syntax**

```bash
# Minute Hour Day Month DayOfWeek Command
# 0-59   0-23  1-31 1-12   0-6(0=Sun)

# Daily at 2 AM
0 2 * * * /usr/local/bin/backup.sh

# Every 15 minutes
*/15 * * * * /usr/local/bin/check.sh

# Monday at 3 AM
0 3 * * 1 /usr/local/bin/weekly.sh

# First of month
0 0 1 * * /usr/local/bin/monthly.sh
```

### **Managing Cron Jobs**

```bash
#!/bin/bash

# List cron jobs
crontab -l

# Edit cron jobs
crontab -e

# Remove cron jobs
crontab -r

# Install cron job from file
crontab /path/to/crontab

# View system cron logs
grep CRON /var/log/syslog
```

## **Part 3: At Command**

### **Scheduling One-Time Tasks**

```bash
#!/bin/bash

# Run command at specific time
at 10:30 AM << 'EOF'
/usr/local/bin/backup.sh
EOF

# Run command at specific date
at 10:30 AM tomorrow << 'EOF'
/usr/local/bin/maintenance.sh
EOF

# List scheduled tasks
atq

# Remove scheduled task
atrm 1
```

## **Part 4: Automated Maintenance**

### **Automated Cleanup Script**

```bash
#!/bin/bash

# Automated daily cleanup
# Add to crontab: 0 2 * * * /usr/local/bin/daily_cleanup.sh

cleanup_old_files() {
    find /tmp -type f -mtime +7 -delete
}

cleanup_logs() {
    find /var/log -type f -name "*.log" -mtime +30 -delete
}

cleanup_backups() {
    find /backup -type f -mtime +60 -delete
}

# Execute
cleanup_old_files
cleanup_logs
cleanup_backups

echo "Cleanup completed" | mail -s "Daily Cleanup Report" admin@example.com
```
