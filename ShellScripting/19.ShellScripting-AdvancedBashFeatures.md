## **Part 1: Advanced Parameter Expansion**

### **Parameter Expansion Techniques**

```bash
#!/bin/bash

# Parameter substitution with defaults
${var-default}        # Use default if unset
${var:-default}       # Use default if unset or empty
${var=default}        # Assign default if unset
${var:=default}       # Assign default if unset or empty
${var+value}          # Use value if set
${var:+value}         # Use value if set and not empty

# Remove suffix/prefix
${var%pattern}        # Remove shortest suffix
${var%%pattern}       # Remove longest suffix
${var#pattern}        # Remove shortest prefix
${var##pattern}       # Remove longest prefix

# Example
filename="document.pdf"
echo "${filename%.pdf}"    # document
echo "${filename##*/}"     # filename from path
```

## **Part 2: Indirect References**

### **Dynamic Variable References**

```bash
#!/bin/bash

# Indirect variable reference
var_name="username"
username="john"
indirect="${!var_name}"
echo "$indirect"  # john

# Dynamic variable creation
for i in {1..3}; do
    declare "var_$i=$i"
done
echo "${var_1} ${var_2} ${var_3}"  # 1 2 3

# Using nameref (Bash 4.3+)
declare -n ref=array_name
ref+=("new element")  # Modifies array_name
```

## **Part 3: Process Substitution**

### **Advanced I/O Redirection**

```bash
#!/bin/bash

# Process substitution
diff <(sort file1.txt) <(sort file2.txt)

# Multiple inputs
paste <(seq 1 5) <(seq 10 14)

# Pipe multiple processes
cat <(echo "line1") <(echo "line2")

# Named pipes simulation
{ echo "file1"; echo "file2"; } > >(sort)
```

## **Part 4: Advanced Control Structures**

### **Complex Flow Control**

```bash
#!/bin/bash

# Case with regex
case "$var" in
    [0-9]*)
        echo "Starts with digit"
        ;;
    [a-z]*)
        echo "Starts with letter"
        ;;
esac

# Array expansion tricks
arr=(${@:2})        # All arguments except first
set -- "${arr[@]}"  # Modify positional parameters

# Using select for menus
select option in "Option 1" "Option 2" "Quit"; do
    case $option in
        "Option 1") echo "You chose 1"; break ;;
        "Option 2") echo "You chose 2"; break ;;
        "Quit") break ;;
    esac
done
```

## **Part 5: Advanced I/O**

### **File Descriptor Management**

```bash
#!/bin/bash

# Redirect to multiple outputs
exec 1> >(tee output.log)
exec 2>&1

# Save and restore file descriptors
exec 3>&1          # Save stdout
exec 1> temp.txt   # Redirect stdout
echo "To file"
exec 1>&3          # Restore stdout

# Read/write to same file
exec 3<> data.txt
echo "written" >&3
read -u 3 line
exec 3>&-          # Close FD
```
