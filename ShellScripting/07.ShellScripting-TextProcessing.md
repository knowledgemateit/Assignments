## **Part 1: Grep - Pattern Matching**

### **Basic Grep Usage**

```bash
#!/bin/bash

# Search for pattern
grep "error" /var/log/syslog

# Search with line numbers
grep -n "error" /var/log/syslog

# Case-insensitive search
grep -i "ERROR" /var/log/syslog

# Invert match (lines without pattern)
grep -v "debug" /var/log/syslog

# Show file names only
grep -l "error" /var/log/*

# Show count of matches
grep -c "error" /var/log/syslog
```

### **Advanced Grep Patterns**

```bash
#!/bin/bash

# Regular expression search
grep "^error" file.txt      # Lines starting with 'error'
grep "error$" file.txt      # Lines ending with 'error'
grep "e.ror" file.txt       # 'e' followed by any char then 'ror'
grep "er*or" file.txt       # 'e' followed by 0+ 'r's then 'or'

# Extended regex with -E
grep -E "(error|warning)" file.txt

# Word boundary search
grep "\<error\>" file.txt

# Multiple patterns
grep -E "error|warning|fatal" file.txt

# Search multiple files
grep "error" *.log
```

### **Grep with Pipes**

```bash
#!/bin/bash

# Filter command output
ps aux | grep "apache"

# Pipeline processing
cat /etc/passwd | grep "admin" | wc -l

# Count specific patterns
ifconfig | grep "inet " | wc -l
```

## **Part 2: Sed - Stream Editing**

### **Basic Sed Operations**

```bash
#!/bin/bash

# Substitute first occurrence on each line
sed 's/old/new/' file.txt

# Substitute all occurrences on each line
sed 's/old/new/g' file.txt

# Substitute on specific line
sed '3s/old/new/' file.txt

# Substitute on line range
sed '2,4s/old/new/g' file.txt

# Save changes to file
sed -i.bak 's/old/new/g' file.txt

# Delete lines
sed '3d' file.txt              # Delete line 3
sed '2,4d' file.txt            # Delete lines 2-4
sed '/pattern/d' file.txt      # Delete lines containing pattern
```

### **Advanced Sed**

```bash
#!/bin/bash

# Print specific lines
sed -n '3p' file.txt

# Extract range
sed -n '10,20p' file.txt

# Multiple operations
sed -e 's/old/new/' -e 's/foo/bar/' file.txt

# Append text
sed '/pattern/a "Appended text"' file.txt

# Insert text
sed '/pattern/i "Inserted text"' file.txt

# Replace entire line
sed '/pattern/c "Replacement line"' file.txt
```

## **Part 3: Awk - Data Extraction and Processing**

### **Basic Awk Usage**

```bash
#!/bin/bash

# Print specific columns
awk '{print $1}' file.txt
awk '{print $1, $3}' file.txt

# Field separator
awk -F: '{print $1, $3}' /etc/passwd

# Print with conditions
awk '{if ($1 == "admin") print}' file.txt

# Using BEGIN and END
awk 'BEGIN {print "Starting"} {print} END {print "Done"}' file.txt

# Line and word count
awk '{lines++; words+=NF} END {print lines, words}' file.txt
```

### **Advanced Awk Patterns**

```bash
#!/bin/bash

# Sum column values
awk '{sum += $2} END {print sum}' data.txt

# Average calculation
awk '{sum += $1; count++} END {print sum/count}' data.txt

# Pattern matching
awk '/error/ {print}' file.txt

# Conditional processing
awk '$3 > 100 {print $1, $3}' file.txt

# Built-in variables
awk 'NR==5 {print}' file.txt           # 5th line
awk 'NF > 0 {print}' file.txt          # Non-empty lines
awk 'END {print NR}' file.txt          # Line count
```

### **Awk String and Math Operations**

```bash
#!/bin/bash

# String concatenation
awk '{print $1 ":" $2}' file.txt

# Length function
awk '{print length($1)}' file.txt

# Substring
awk '{print substr($1, 1, 3)}' file.txt

# Math operations
awk '{print $1 * 2}' file.txt
awk '{print $1 + $2}' file.txt
awk '{print sqrt($1)}' file.txt
```

## **Part 4: Other Text Processing Tools**

### **Cut - Extract Columns**

```bash
#!/bin/bash

# Extract by character position
cut -c 1-5 file.txt

# Extract by field
cut -d: -f1,3 /etc/passwd

# Extract range of fields
cut -d, -f2-4 data.csv

# Complement (all except specified)
cut -d: -f1 --complement /etc/passwd
```

### **Paste - Merge Files**

```bash
#!/bin/bash

# Merge lines from files
paste file1.txt file2.txt

# Custom delimiter
paste -d: file1.txt file2.txt

# Merge columns
paste -d, col1.txt col2.txt col3.txt
```

### **Sort - Order Data**

```bash
#!/bin/bash

# Sort alphabetically
sort file.txt

# Sort numerically
sort -n numbers.txt

# Sort reverse
sort -r file.txt

# Sort by field
sort -k2 -t: /etc/passwd

# Remove duplicates while sorting
sort -u file.txt

# Sort in place
sort -o file.txt file.txt
```

### **Uniq - Remove Duplicates**

```bash
#!/bin/bash

# Remove consecutive duplicates
uniq file.txt

# Count occurrences
uniq -c file.txt

# Show only duplicates
uniq -d file.txt

# Show only unique lines
uniq -u file.txt
```

## **Part 5: Combining Tools**

### **Pipeline Processing**

```bash
#!/bin/bash

# Find errors, count by type
grep "ERROR" /var/log/app.log | \
    awk '{print $2}' | \
    sort | \
    uniq -c | \
    sort -rn

# Extract and format data
cat /etc/passwd | \
    cut -d: -f1,3 | \
    grep -E ":10" | \
    awk -F: '{print $1}'

# Process CSV file
cat data.csv | \
    cut -d, -f1,2,3 | \
    sed 's/^/"/g' | \
    paste -sd, -
```

## **Troubleshooting Guide**

| Issue | Solution |
|-------|----------|
| Grep not finding patterns | Check case sensitivity, use `-i` for case-insensitive |
| Sed not replacing | Escape special characters, use different delimiter |
| Awk field not extracting | Check field separator with `-F`, use correct column number |
| Pipeline failing | Check each command individually first |
| Performance slow | Use appropriate tools, consider file size |

---

## **Best Practices**

1. **Test commands on small files** before processing large files
2. **Use quotes** to prevent shell expansion
3. **Escape special characters** in patterns
4. **Use appropriate tools** for the task
5. **Combine tools efficiently** in pipelines
6. **Use -n flag** with sed to improve performance
7. **Use consistent field separators** when combining tools
8. **Check output** before using in pipelines
9. **Document complex pipelines** with comments
10. **Save original files** before modifying with sed -i
