## **Part 1: Install Ansible**

### **On Windows (Using WSL2 or Git Bash)**

1. Install via Python package manager:
   ```bash
   pip install ansible
   ```

2. Verify installation:
   ```bash
   ansible --version
   ```
   **Expected Output:**
   ```
   ansible 2.10.0
   ```

### **Alternative: Using Windows Subsystem for Linux (WSL2)**

1. Open PowerShell as Administrator
2. Install WSL2:
   ```powershell
   wsl --install
   ```

3. Install Ansible in WSL2:
   ```bash
   sudo apt update
   sudo apt install ansible
   ```

## **Part 2: Set Up Ansible Environment**

### **Step 1: Create Project Structure**

1. Create project directory:
   ```bash
   mkdir ~/ansible-project
   cd ~/ansible-project
   ```

2. Create subdirectories:
   ```bash
   mkdir -p playbooks roles inventories group_vars host_vars
   ```

### **Step 2: Create Inventory File**

1. Create `inventories/hosts.ini`:
   ```ini
   [local]
   localhost ansible_connection=local ansible_python_interpreter=/usr/bin/python3

   [webservers]
   web1.example.com
   web2.example.com
   
   [databases]
   db1.example.com
   db2.example.com
   
   [all:vars]
   ansible_user=admin
   ansible_ssh_key_file=~/.ssh/id_rsa
   ```

### **Step 3: Create Ansible Configuration**

1. Create `ansible.cfg`:
   ```ini
   [defaults]
   inventory = inventories/hosts.ini
   host_key_checking = False
   deprecation_warnings = False
   log_path = ./ansible.log
   ```

## **Part 3: Verify Connectivity**

### **Step 1: Ping Hosts**

1. Test connectivity to localhost:
   ```bash
   ansible all -i inventories/hosts.ini -m ping
   ```
   **Expected Output:**
   ```
   localhost | SUCCESS => {
       "changed": false,
       "ping": "pong"
   }
   ```

### **Step 2: Gather Facts**

1. Gather system information:
   ```bash
   ansible localhost -m setup
   ```

## **Part 4: Create Your First Playbook**

### **Step 1: Simple Task Playbook**

1. Create `playbooks/01-basic.yml`:
   ```yaml
   ---
   - name: Basic Ansible Playbook
     hosts: localhost
     gather_facts: yes
     tasks:
       - name: Display hostname
         debug:
           msg: "Hostname is {{ ansible_hostname }}"
       
       - name: Display OS family
         debug:
           msg: "OS Family is {{ ansible_os_family }}"
       
       - name: Create a directory
         file:
           path: /tmp/ansible-demo
           state: directory
           mode: '0755'
       
       - name: Create a file
         file:
           path: /tmp/ansible-demo/test.txt
           state: touch
   ```

### **Step 2: Run the Playbook**

1. Execute playbook:
   ```bash
   ansible-playbook playbooks/01-basic.yml
   ```
   **Expected Output:**
   ```
   PLAY [Basic Ansible Playbook]
   ...
   TASK [Display hostname]
   ok: [localhost] => {
       "msg": "Hostname is mycomputer"
   }
   ```

## **Part 5: Package Installation Playbook**

### **Step 1: Create Installation Playbook**

1. Create `playbooks/02-install-packages.yml`:
   ```yaml
   ---
   - name: Install and Configure Packages
     hosts: localhost
     become: yes
     tasks:
       - name: Update package cache
         apt:
           update_cache: yes
           cache_valid_time: 3600
         when: ansible_os_family == 'Debian'
       
       - name: Install required packages
         apt:
           name:
             - curl
             - wget
             - git
             - python3-pip
           state: present
         when: ansible_os_family == 'Debian'
       
       - name: Install Python packages
         pip:
           name:
             - requests
             - flask
             - jinja2
           state: present
       
       - name: Verify git installation
         command: git --version
         register: git_version
       
       - name: Display git version
         debug:
           msg: "{{ git_version.stdout }}"
   ```

### **Step 2: Run Installation Playbook**

1. Execute playbook:
   ```bash
   ansible-playbook playbooks/02-install-packages.yml
   ```

## **Part 6: Configuration Management Playbook**

### **Step 1: Create Configuration Playbook**

1. Create `playbooks/03-configure-app.yml`:
   ```yaml
   ---
   - name: Configure Application
     hosts: localhost
     vars:
       app_user: appuser
       app_dir: /opt/myapp
       app_port: 8080
     tasks:
       - name: Create application user
         user:
           name: "{{ app_user }}"
           shell: /bin/bash
           createhome: yes
           state: present
         when: ansible_os_family == 'Debian'
       
       - name: Create application directory
         file:
           path: "{{ app_dir }}"
           state: directory
           owner: "{{ app_user }}"
           group: "{{ app_user }}"
           mode: '0755'
       
       - name: Create application config file
         copy:
           content: |
             # Application Configuration
             SERVER_PORT={{ app_port }}
             DEBUG=false
             LOG_LEVEL=info
           dest: "{{ app_dir }}/config.ini"
           owner: "{{ app_user }}"
           group: "{{ app_user }}"
           mode: '0644'
       
       - name: Create systemd service file
         copy:
           content: |
             [Unit]
             Description=My Application
             After=network.target
             
             [Service]
             Type=simple
             User={{ app_user }}
             WorkingDirectory={{ app_dir }}
             ExecStart=/usr/bin/python3 app.py
             Restart=always
             
             [Install]
             WantedBy=multi-user.target
           dest: /etc/systemd/system/myapp.service
         become: yes
       
       - name: Enable and start service
         systemd:
           name: myapp
           enabled: yes
           state: started
         become: yes
   ```

## **Part 7: Variables and Templates**

### **Step 1: Create Variable Files**

1. Create `group_vars/localhost.yml`:
   ```yaml
   ---
   # Database Configuration
   db_host: localhost
   db_port: 5432
   db_name: myapp
   db_user: dbuser
   
   # Application Configuration
   app_version: 1.0.0
   app_environment: production
   
   # System Configuration
   timezone: UTC
   ntp_servers:
     - 0.pool.ntp.org
     - 1.pool.ntp.org
   ```

### **Step 2: Create Template Playbook**

1. Create `playbooks/04-templates.yml`:
   ```yaml
   ---
   - name: Deploy Configuration Files
     hosts: localhost
     tasks:
       - name: Generate nginx config from template
         template:
           src: nginx.conf.j2
           dest: /tmp/nginx.conf
           mode: '0644'
       
       - name: Display generated config
         debug:
           msg: "{{ lookup('file', '/tmp/nginx.conf') }}"
   ```

### **Step 3: Create Template File**

1. Create `templates/nginx.conf.j2`:
   ```jinja2
   # Generated by Ansible
   server {
       listen {{ app_port }};
       server_name _;
       
       location / {
           proxy_pass http://localhost:8080;
       }
   }
   ```

## **Part 8: Handlers and Notifications**

### **Step 1: Create Playbook with Handlers**

1. Create `playbooks/05-handlers.yml`:
   ```yaml
   ---
   - name: Configuration with Handlers
     hosts: localhost
     handlers:
       - name: Restart nginx
         systemd:
           name: nginx
           state: restarted
         become: yes
       
       - name: Reload firewall
         command: ufw reload
         become: yes
     
     tasks:
       - name: Install nginx
         apt:
           name: nginx
           state: present
         become: yes
       
       - name: Copy nginx config
         copy:
           content: |
             server {
                 listen 80;
                 server_name _;
             }
           dest: /etc/nginx/sites-available/default
         become: yes
         notify: Restart nginx
   ```

## **Part 9: Error Handling**

### **Step 1: Create Playbook with Error Handling**

1. Create `playbooks/06-error-handling.yml`:
   ```yaml
   ---
   - name: Error Handling Examples
     hosts: localhost
     tasks:
       - name: Task that might fail
         command: /bin/false
         ignore_errors: yes
       
       - name: Check if file exists
         stat:
           path: /tmp/test.txt
         register: file_stat
       
       - name: Create file if not exists
         file:
           path: /tmp/test.txt
           state: touch
         when: not file_stat.stat.exists
       
       - name: Task with retry
         command: echo "Success"
         retries: 3
         delay: 2
         until: true
   ```

## **Part 10: Running Playbooks**

### **Execute Playbooks**

1. Run all playbooks:
   ```bash
   ansible-playbook playbooks/*.yml -v
   ```

2. Run specific playbook:
   ```bash
   ansible-playbook playbooks/01-basic.yml
   ```

3. Run with extra variables:
   ```bash
   ansible-playbook playbooks/03-configure-app.yml -e "app_port=9000"
   ```

4. Dry run (check mode):
   ```bash
   ansible-playbook playbooks/01-basic.yml --check
   ```
