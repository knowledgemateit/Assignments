## **Part 1: Install Apache Tomcat**

### **Step 1: Download Tomcat**

1. Visit **https://tomcat.apache.org/download-10.cgi**
2. Download **Core** distribution (zip file)
3. Extract to `C:\tomcat`

### **Step 2: Set JAVA_HOME**

1. Set Java environment variable:
   ```powershell
   [Environment]::SetEnvironmentVariable("JAVA_HOME", "C:\Program Files\Java\jdk-17", "Machine")
   $env:JAVA_HOME
   ```

### **Step 3: Start Tomcat**

1. Open Command Prompt and navigate to Tomcat:
   ```powershell
   cd C:\tomcat\bin
   catalina.bat start
   ```

2. Or run startup script:
   ```powershell
   startup.bat
   ```

### **Step 4: Verify Installation**

1. Open browser: **http://localhost:8080**
2. You should see the Tomcat welcome page

### **Step 5: Stop Tomcat**

1. Run shutdown script:
   ```powershell
   shutdown.bat
   ```

## **Part 2: Tomcat Directory Structure**

Understand Tomcat directory layout:
```
C:\tomcat\
├── bin/                    # Startup and shutdown scripts
├── conf/                   # Configuration files
│   ├── server.xml         # Main configuration
│   ├── tomcat-users.xml   # User authentication
│   └── context.xml        # Application context
├── lib/                    # JAR libraries
├── logs/                   # Log files
├── temp/                   # Temporary files
├── webapps/               # Deployed applications
│   ├── ROOT/              # Root application
│   ├── manager/           # Manager application
│   └── host-manager/      # Host manager application
└── work/                  # Working directory
```

## **Part 3: Tomcat Configuration**

### **Step 1: Configure server.xml**

1. Open `C:\tomcat\conf\server.xml`:
   ```xml
   <!-- Set Tomcat port -->
   <Connector port="8080" protocol="HTTP/1.1"
              connectionTimeout="20000"
              redirectPort="8443" />
   
   <!-- HTTPS Connector -->
   <Connector port="8443" protocol="org.apache.coyote.http11.Http11NioProtocol"
              maxThreads="200"
              scheme="https" secure="true" sslProtocol="TLS" />
   
   <!-- Shutdown port -->
   <Server port="8005" shutdown="SHUTDOWN">
   ```

### **Step 2: Configure Tomcat Users**

1. Open `C:\tomcat\conf\tomcat-users.xml`:
   ```xml
   <tomcat-users>
       <role rolename="manager-gui"/>
       <role rolename="admin-gui"/>
       <user username="admin" password="admin123" roles="manager-gui,admin-gui"/>
       <user username="manager" password="manager123" roles="manager-gui"/>
   </tomcat-users>
   ```

### **Step 3: Configure Manager Application**

1. Create `C:\tomcat\conf\Catalina\localhost\manager.xml`:
   ```xml
   <Context
       privileged="true"
       antiResourceLocking="false"
       docBase="${catalina.home}/webapps/manager">
   </Context>
   ```

## **Part 4: Deploy Applications**

### **Option 1: Copy WAR File to webapps**

1. Create a sample WAR file or download one
2. Copy to `C:\tomcat\webapps\`
   ```powershell
   Copy-Item "C:\my-app.war" "C:\tomcat\webapps\"
   ```

3. Start Tomcat - it auto-deploys the WAR

### **Option 2: Use Manager Application**

1. Start Tomcat
2. Visit **http://localhost:8080/manager/html**
3. Login with your configured credentials
4. In **Deploy section**:
   - Select WAR file to upload
   - Enter application name
   - Click **Deploy**

### **Option 3: Programmatic Deployment**

1. Create `C:\tomcat\conf\Catalina\localhost\myapp.xml`:
   ```xml
   <Context 
       docBase="C:\path\to\myapp.war" 
       path="/myapp" 
       reloadable="true">
   </Context>
   ```

## **Part 5: Create and Deploy a Sample Application**

### **Step 1: Create a Simple JSP Application**

1. Create directory structure:
   ```powershell
   mkdir C:\sample-app\webapps\ROOT
   mkdir C:\sample-app\webapps\ROOT\WEB-INF
   ```

2. Create `index.jsp`:
   ```jsp
   <%@ page contentType="text/html; charset=UTF-8" %>
   <!DOCTYPE html>
   <html>
   <head>
       <title>Tomcat Application</title>
       <style>
           body { font-family: Arial, sans-serif; }
           .container { max-width: 800px; margin: 50px auto; }
       </style>
   </head>
   <body>
       <div class="container">
           <h1>Welcome to Tomcat!</h1>
           <p>Server Time: <%= new java.util.Date() %></p>
           <p>Tomcat Server Name: <%= request.getServerName() %></p>
           <p>Server Port: <%= request.getServerPort() %></p>
       </div>
   </body>
   </html>
   ```

### **Step 2: Create web.xml**

1. Create `C:\sample-app\webapps\ROOT\WEB-INF\web.xml`:
   ```xml
   <?xml version="1.0" encoding="UTF-8"?>
   <web-app version="4.0"
            xmlns="http://xmlns.jcp.org/xml/ns/javaee"
            xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
            xsi:schemaLocation="http://xmlns.jcp.org/xml/ns/javaee
                                http://xmlns.jcp.org/xml/ns/javaee/web-app_4_0.xsd">
       <display-name>Sample Application</display-name>
       <welcome-file-list>
           <welcome-file>index.jsp</welcome-file>
       </welcome-file-list>
   </web-app>
   ```

### **Step 3: Package as WAR**

1. Create WAR file:
   ```powershell
   cd C:\sample-app\webapps
   jar -cvf sample-app.war ROOT/
   ```

### **Step 4: Deploy to Tomcat**

1. Copy to Tomcat webapps:
   ```powershell
   Copy-Item "C:\sample-app\webapps\sample-app.war" "C:\tomcat\webapps\"
   ```

2. Start Tomcat
3. Visit: **http://localhost:8080/sample-app**

## **Part 6: Tomcat Monitoring**

### **Step 1: View Manager Dashboard**

1. Visit **http://localhost:8080/manager/html**
2. Monitor:
   - Deployed applications
   - Memory usage
   - Thread information
   - Active sessions

### **Step 2: View Logs**

1. Check Tomcat logs:
   ```powershell
   Get-Content "C:\tomcat\logs\catalina.log"
   Get-Content "C:\tomcat\logs\localhost.log"
   ```

### **Step 3: Configure Logging**

1. Edit `C:\tomcat\conf\logging.properties`:
   ```properties
   1catalina.org.apache.juli.FileHandler.level = FINE
   1catalina.org.apache.juli.FileHandler.directory = ${catalina.base}/logs
   1catalina.org.apache.juli.FileHandler.prefix = catalina.
   1catalina.org.apache.juli.FileHandler.maxDays = 90
   ```

## **Part 7: Performance Tuning**

### **Step 1: Configure JVM Settings**

1. Create `C:\tomcat\bin\setenv.bat`:
   ```batch
   set JAVA_OPTS=-Xms512m -Xmx1024m -XX:+UseG1GC
   set JAVA_OPTS=%JAVA_OPTS% -Djava.awt.headless=true
   ```

### **Step 2: Configure Connection Pool**

1. Edit `server.xml`:
   ```xml
   <Resource name="jdbc/mydb"
             auth="Container"
             type="javax.sql.DataSource"
             driverClassName="com.mysql.jdbc.Driver"
             url="jdbc:mysql://localhost:3306/mydb"
             username="root"
             password="password"
             maxTotal="20"
             maxIdle="10"
             maxWaitMillis="5000" />
   ```

### **Step 3: Configure Thread Pool**

1. Edit connector in `server.xml`:
   ```xml
   <Connector port="8080" protocol="HTTP/1.1"
              connectionTimeout="20000"
              maxThreads="200"
              minSpareThreads="10"
              redirectPort="8443" />
   ```

## **Part 8: Security Configuration**

### **Step 1: Enable HTTPS**

1. Generate SSL certificate:
   ```powershell
   keytool -genkey -alias tomcat -keyalg RSA -keystore C:\tomcat\keystore.jks
   ```

2. Configure SSL in `server.xml`:
   ```xml
   <Connector port="8443" protocol="HTTP/1.1" SSLEnabled="true"
              scheme="https" secure="true"
              keystoreFile="C:/tomcat/keystore.jks"
              keystorePass="changeit" />
   ```

### **Step 2: Redirect HTTP to HTTPS**

1. Add to application `web.xml`:
   ```xml
   <security-constraint>
       <web-resource-collection>
           <web-resource-name>Entire Application</web-resource-name>
           <url-pattern>/*</url-pattern>
       </web-resource-collection>
       <user-data-constraint>
           <transport-guarantee>CONFIDENTIAL</transport-guarantee>
       </user-data-constraint>
   </security-constraint>
   ```

## **Part 9: Clustering (Advanced)**

### **Step 1: Configure Session Clustering**

1. Edit `server.xml` to enable clustering:
   ```xml
   <Engine name="Catalina" defaultHost="localhost" jvmRoute="tomcat1">
       <Cluster className="org.apache.catalina.ha.tcp.SimpleTcpCluster" />
   </Engine>
   ```

2. In application context:
   ```xml
   <Context distributable="true">
   </Context>
   ```
