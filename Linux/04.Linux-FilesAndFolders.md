## **Part 1: File and Folder Basics**

### **File Naming Conventions**

```bash
# Valid filenames
myfile.txt
my-file.txt
my_file.txt
myFile123.txt
.hidden-file

# Avoid in filenames
Spaces in filename          # Use underscores or hyphens
Special characters (*?<>)   # Use letters, numbers, -, _
```

### **Absolute vs Relative Paths**

```bash
# Absolute path (starts with /)
/home/username/Documents/file.txt

# Relative path (from current location)
Documents/file.txt

# Current directory
.

# Parent directory
..
```

## **Part 2: Creating Files and Folders**

### **Create Files**

```bash
# Method 1: touch (creates empty file)
touch myfile.txt
touch file1.txt file2.txt file3.txt

# Method 2: echo with redirection
echo "Hello, Linux!" > greeting.txt

# Method 3: cat with here-document
cat > document.txt << EOF
This is a
multi-line
document
EOF

# Method 4: nano or vim (edit while creating)
nano newfile.txt
```

### **Create Folders**

```bash
# Create single folder
mkdir myfolder

# Create nested folders
mkdir -p documents/projects/python
# -p creates parent directories if needed

# Create multiple folders
mkdir folder1 folder2 folder3

# View created folders
tree documents 2>/dev/null || ls -R documents
```

## **Part 3: Listing Files and Folders**

### **ls Command Variations**

```bash
# List files (default)
ls

# List with details
ls -l

# List all files (including hidden)
ls -a

# List all with details
ls -la

# List recursively
ls -R

# List with human-readable sizes
ls -lh

# List sorted by size
ls -lhS

# List sorted by time
ls -lht

# List with directories first
ls -l --group-directories-first

# List only directories
ls -d */
```

### **Understanding ls -l Output**

```
-rw-r--r-- 1 user group 1234 Jan 15 10:30 filename.txt

-         : File type (- = regular file, d = directory, l = link)
rw-       : Owner permissions (read, write, execute)
r--       : Group permissions
r--       : Other permissions
1         : Number of hard links
user      : Owner username
group     : Group name
1234      : File size in bytes
Jan 15    : Modification date
10:30     : Modification time
filename  : Filename
```

## **Part 4: Copying Files and Folders**

### **cp Command**

```bash
# Copy file
cp source.txt destination.txt

# Copy with interactive prompt (if exists)
cp -i source.txt destination.txt

# Copy and preserve permissions
cp -p source.txt destination.txt

# Copy recursively (folder)
cp -r source-folder/ destination-folder/

# Copy multiple files
cp file1.txt file2.txt file3.txt destination-folder/

# Copy with verbose output
cp -v source.txt destination.txt

# Combine options
cp -riv source-folder/ destination-folder/
```

## **Part 5: Moving and Renaming Files and Folders**

### **mv Command**

```bash
# Move file
mv source.txt destination-folder/

# Rename file
mv old-name.txt new-name.txt

# Move and rename
mv source.txt destination-folder/new-name.txt

# Move folder recursively
mv source-folder/ destination-folder/

# Move multiple files
mv file1.txt file2.txt destination-folder/

# Interactive mode (prompt if overwrite)
mv -i source.txt destination.txt

# Force move (no prompt)
mv -f source.txt destination.txt

# Verbose output
mv -v source.txt destination.txt
```

## **Part 6: Deleting Files and Folders**

### **rm Command**

```bash
# Delete file
rm myfile.txt

# Delete multiple files
rm file1.txt file2.txt file3.txt

# Interactive mode (confirm before delete)
rm -i myfile.txt

# Force delete (no confirmation)
rm -f myfile.txt

# Delete folder and contents
rm -r myfolder/

# Verbose output
rm -v myfile.txt

# Combine options
rm -riv myfolder/
```

### **rmdir Command**

```bash
# Delete empty directory only
rmdir empty-folder/

# Remove parent directories if empty
rmdir -p path/to/empty/folder/
```

## **Part 8: File Links**

### **Hard Links**

```bash
# Create hard link
ln source.txt hardlink.txt

# Hard links point to same inode
ls -i source.txt hardlink.txt

# Both files are equal
cat source.txt
cat hardlink.txt

# Delete original - link still works
rm source.txt
cat hardlink.txt              # Still accessible
```

### **Symbolic Links (Symlinks)**

```bash
# Create symbolic link
ln -s source.txt symlink.txt

# View symlink
ls -l symlink.txt            # Shows arrow

# Check symlink target
readlink symlink.txt

# Delete original - symlink breaks
rm source.txt
cat symlink.txt              # Error: No such file

# Create symlink to folder
ln -s /full/path/folder ~/mylink
```

### **Difference: Hard Link vs Symlink**

| Feature | Hard Link | Symlink |
|---------|-----------|---------|
| Storage | Same inode | New inode |
| Across filesystems | No | Yes |
| Works if original deleted | Yes | No |
| Target type | File only | File or folder |
| Speed | Faster | Slightly slower |

## **Part 9: Finding Files**

### **find Command**

```bash
# Find files by name
find . -name "*.txt"

# Find files by type
find . -type f              # Regular files
find . -type d              # Directories
find . -type l              # Symlinks

# Find by size
find . -size +1M            # Larger than 1MB
find . -size -100k          # Smaller than 100KB

# Find by modification time
find . -mtime -7            # Modified within 7 days
find . -mtime +30           # Modified older than 30 days

# Find by permissions
find . -perm 644

# Find and execute command
find . -name "*.txt" -exec cat {} \;

# Find with case-insensitive
find . -iname "*.TXT"
```

### **locate Command**

```bash
# Locate file quickly (uses database)
locate myfile.txt

# Update database
updatedb

# Locate with multiple matches
locate -i filename          # Case insensitive
```

### **which and whereis Commands**

```bash
# Find command in PATH
which ls
which python

# Find command location and manual
whereis ls
```

## **Part 10: Working with Archives**

### **Compress Files (gzip)**

```bash
# Compress single file
gzip myfile.txt             # Creates myfile.txt.gz

# Keep original
gzip -k myfile.txt

# Decompress
gunzip myfile.txt.gz
gzip -d myfile.txt.gz

# Compress folder
tar -czf archive.tar.gz folder/
```

### **Zip Format**

```bash
# Create zip file
zip myfile.zip myfile.txt

# Add files to zip
zip myfile.zip file1.txt file2.txt

# Zip entire folder
zip -r myfolder.zip folder/

# Extract zip
unzip myfile.zip

# List contents
unzip -l myfile.zip
```

### **Tar Archives**

```bash
# Create tar archive
tar -cf archive.tar folder/

# Create tar with compression (gzip)
tar -czf archive.tar.gz folder/

# Create tar with compression (bzip2)
tar -cjf archive.tar.bz2 folder/

# Extract tar
tar -xf archive.tar

# Extract tar.gz
tar -xzf archive.tar.gz

# Extract tar.bz2
tar -xjf archive.tar.bz2

# List contents
tar -tzf archive.tar.gz

# Extract to specific directory
tar -xzf archive.tar.gz -C /destination/
```
