## **Part 1: Linux Filesystem Hierarchy Standard (FHS)**

### **Overview**

FHS defines standard directory structure for Linux. Root directory is `/`.

```bash
# View root directory
cd /
ls -la
```

---

## **Part 2: Essential Root Level Directories**

### **Directory Reference**

| Directory | Purpose | Content |
|-----------|---------|---------|
| `/bin` | Essential user binaries | Commands, programs |
| `/sbin` | System binaries | Admin commands |
| `/boot` | Boot files | Kernel, bootloader |
| `/dev` | Device files | Hardware devices |
| `/etc` | Configuration files | System configs |
| `/home` | User home directories | User files |
| `/lib` | Libraries | System libraries |
| `/media` | Removable media | USB, CD/DVD |
| `/mnt` | Mount points | Temp mounts |
| `/opt` | Optional packages | Third-party software |
| `/proc` | Process info | Virtual filesystem |
| `/root` | Root user home | Admin's home |
| `/run` | Runtime data | Temporary files |
| `/srv` | Service data | Web content |
| `/sys` | System info | Kernel interface |
| `/tmp` | Temporary files | Temp storage |
| `/usr` | User programs | Applications |
| `/var` | Variable data | Logs, caches |

## **Part 3: Exploring Root Directories**

### **`/bin` - User Binaries**

Essential commands available to all users.

```bash
# List binaries in /bin
ls /bin

# View common programs
ls /bin | head -20

# Check if command exists in /bin
which bash
which ls
which cat
```

**Important binaries:**
```
bash, sh, cat, ls, cp, mv, rm, grep, sed, awk
chmod, chown, touch, mkdir, rmdir, pwd, cd
```

### **`/sbin` - System Binaries**

Administrative commands, usually require root.

```bash
# List system binaries
ls /sbin

# These typically require sudo
sudo /sbin/shutdown --help
sudo /sbin/reboot --help
```

### **`/boot` - Boot Files**

Contains bootloader and kernel files.

```bash
# List boot files
ls -la /boot

# View kernel files
ls /boot/vmlinuz*

# View initramfs
ls /boot/initrd*
```

### **`/dev` - Device Files**

All hardware devices represented as files.

```bash
# List device files
ls /dev

# Character devices (terminal, null, zero)
ls -la /dev/tty*
ls -la /dev/null
ls -la /dev/zero

# Block devices (disks, USB)
ls -la /dev/sd*
ls -la /dev/nvme*

# Loop devices
ls -la /dev/loop*

# Random number generators
ls -la /dev/random
ls -la /dev/urandom
```

### **`/etc` - Configuration Files**

System and application configuration files (text-based).

```bash
# List important config files
ls /etc

# System configuration
cat /etc/hostname              # System hostname
cat /etc/hosts                 # Host file
cat /etc/fstab                 # Filesystem mount table
cat /etc/sudoers               # Sudo permissions
cat /etc/shadow                # Encrypted passwords
cat /etc/passwd                # User database
cat /etc/group                 # Group database

# Application configs
ls /etc/ssh/                   # SSH configuration
ls /etc/apache2/               # Apache configuration
ls /etc/mysql/                 # MySQL configuration
ls /etc/nginx/                 # Nginx configuration
```

### **`/home` - User Home Directories**

Each user has a home directory.

```bash
# List home directories
ls /home

# Your home directory
echo $HOME

# Navigate to home
cd ~

# View hidden files in home
ls -la ~

# Important home subdirectories
ls -la ~/.ssh                  # SSH keys
ls -la ~/.bashrc               # Bash configuration
ls -la ~/.bash_history         # Command history
```

### **`/lib` - System Libraries**

Shared libraries needed by programs.

```bash
# View libraries
ls /lib

# Check library dependencies
ldd /bin/ls

# Find specific library
ls /lib/x86_64-linux-gnu/ | grep libc
```

### **`/proc` - Process Information**

Virtual filesystem showing running processes and system info.

```bash
# List processes
ls /proc | head -20

# View process-specific info
ls /proc/1                     # Init process
cat /proc/1/cmdline            # Process command line
cat /proc/1/status             # Process status

# System information
cat /proc/cpuinfo              # CPU information
cat /proc/meminfo              # Memory information
cat /proc/uptime               # System uptime
cat /proc/version              # Kernel version
cat /proc/filesystems          # Supported filesystems
cat /proc/mounts               # Mounted filesystems
```

### **`/root` - Root User Home**

Home directory for root user (system administrator).

```bash
# View root home (requires sudo)
sudo ls -la /root

# Root's bash history
sudo cat /root/.bash_history | tail -20
```

### **`/tmp` - Temporary Files**

World-writable directory for temporary files.

```bash
# List temp files
ls -la /tmp

# Create temporary file
echo "test" > /tmp/myfile.txt

# View file
cat /tmp/myfile.txt

# Clean up
rm /tmp/myfile.txt
```

### **`/var` - Variable Data**

Contains files that change during system operation.

```bash
# System logs
ls /var/log
cat /var/log/syslog            # System log
cat /var/log/auth.log          # Authentication log
cat /var/log/apt/history.log   # Package history

# Caches
ls /var/cache

# Mail
ls /var/mail

# Temporary runtime data
ls /var/run
```

### **`/usr` - User Programs**

User applications and utilities (not essential for boot).

```bash
# Installed programs
ls /usr/bin
ls /usr/sbin

# Libraries
ls /usr/lib

# Documentation
ls /usr/share/doc
cat /usr/share/doc/bash/README

# Man pages
ls /usr/share/man

# Local programs (user-installed)
ls /usr/local/bin
```

### **`/opt` - Optional Packages**

Third-party software packages.

```bash
# List optional packages
ls /opt

# Example: VirtualBox installation
ls /opt/VirtualBox

# User-specific optional packages
ls /opt/*/bin
```

## **Part 4: Understanding File System Layout**

### **Essential vs Non-Essential**

```bash
# Essential - needed for boot
/bin, /sbin, /etc, /lib, /boot, /dev

# Non-essential - can be on separate partition
/home, /usr, /var, /opt, /tmp
```

### **Single vs Multi-Partition**

```bash
# View current mounts and partitions
df -h

# View filesystem types
df -T

# Show partition details
lsblk

# Mount information
cat /etc/fstab
```

## **Part 7: Directory Permissions**

### **View Directory Permissions**

```bash
# List with permissions
ls -ld /                       # Root directory
ls -ld /bin                    # /bin permissions
ls -ld /home                   # /home permissions
ls -ld /tmp                    # /tmp permissions
ls -ld /root                   # /root permissions

# Understand permissions
# drwxr-xr-x = directory, rwx owner, r-x group, r-x others
# r = read (4)
# w = write (2)
# x = execute (1)
```
