## **Part 1: Package Management Overview**

### **What is Package Management?**

Package managers handle:
- Software installation
- Dependency resolution
- Version management
- Software removal
- System updates

### **Package Manager Types**

**Debian-based (Ubuntu, Debian, Linux Mint):**
- APT (Advanced Package Tool)
- dpkg (low-level)

**Red Hat-based (CentOS, RHEL, Fedora):**
- YUM (Yellowdog Updater Modified)
- DNF (Dandified YUM - newer)
- rpm (low-level)

**Independent:**
- Pacman (Arch Linux)
- Zypper (openSUSE)

## **Part 2: APT (Advanced Package Tool)**

### **Check Your System**

```bash
# Check if APT is available
which apt

# Show APT version
apt --version
```

### **Update Package Lists**

```bash
# Update package list from repositories
sudo apt update

# This downloads list of available packages
# Does NOT upgrade packages
```

### **Upgrade Packages**

```bash
# Upgrade all installed packages
sudo apt upgrade

# Full upgrade (may remove packages)
sudo apt full-upgrade

# Upgrade specific package
sudo apt upgrade package-name
```

### **Install Packages**

```bash
# Install single package
sudo apt install package-name

# Install multiple packages
sudo apt install package1 package2 package3

# Install without confirmation
sudo apt install -y package-name

# Install and remove conflicts
sudo apt install --fix-missing package-name
```

### **Search for Packages**

```bash
# Search for package
apt search package-name

# Search with regex
apt search "^python"

# Show package information
apt info package-name

# Show package details
apt show package-name

# List all available packages
apt list --all-versions
```

### **Remove Packages**

```bash
# Remove package (keeps configuration)
sudo apt remove package-name

# Remove package and configuration files
sudo apt purge package-name

# Remove unused dependencies
sudo apt autoremove

# Clean package cache
sudo apt autoclean
sudo apt clean
```

### **List Installed Packages**

```bash
# List all installed packages
apt list --installed

# Count installed packages
apt list --installed | wc -l

# Show specific package
apt list --installed | grep package-name
```

## **Part 3: YUM/DNF (Red Hat-based)**

### **Check Your System**

```bash
# Check if DNF is available
which dnf

# Check YUM
which yum

# Show version
dnf --version
```

### **Update Package Lists and System**

```bash
# Update package cache (automatic in DNF)
sudo dnf check-update

# Update all packages
sudo dnf upgrade

# Update specific package
sudo dnf upgrade package-name
```

### **Install Packages**

```bash
# Install single package
sudo dnf install package-name

# Install multiple packages
sudo dnf install package1 package2

# Install without confirmation
sudo dnf install -y package-name
```

### **Search and Information**

```bash
# Search for package
sudo dnf search package-name

# Show package information
sudo dnf info package-name

# List available packages
sudo dnf list available

# List installed packages
sudo dnf list installed
```

### **Remove Packages**

```bash
# Remove package
sudo dnf remove package-name

# Remove package and dependencies
sudo dnf remove -y package-name

# Remove unused dependencies
sudo dnf autoremove
```

### **Clean Cache**

```bash
# Clean all cache
sudo dnf clean all

# Clean package cache only
sudo dnf clean packages
```

## **Part 4: dpkg (Low-level Package Manager)**

For Debian-based systems (what APT uses underneath).

```bash
# List all installed packages
dpkg -l

# List specific package
dpkg -l | grep package-name

# Check if package is installed
dpkg -l | grep ^ii | grep package-name

# Show package status
dpkg -s package-name

# List files installed by package
dpkg -L package-name

# Find which package owns file
dpkg -S /path/to/file

# Install .deb file locally
sudo dpkg -i package.deb

# Remove package
sudo dpkg -r package-name

# Full removal (purge)
sudo dpkg -P package-name
```

---

## **Part 5: rpm (Low-level Package Manager)**

For Red Hat-based systems (what YUM/DNF uses underneath).

```bash
# List all installed packages
rpm -qa

# List specific package
rpm -qa | grep package-name

# Show package information
rpm -qi package-name

# List files installed by package
rpm -ql package-name

# Find which package owns file
rpm -qf /path/to/file

# Install .rpm file locally
sudo rpm -i package.rpm

# Remove package
sudo rpm -e package-name

# Verify installed package
rpm -V package-name
```

## **Part 6: System Updates**

### **Ubuntu/Debian**

```bash
# Update package lists
sudo apt update

# Upgrade all packages
sudo apt upgrade

# Update specific packages
sudo apt upgrade package-name

# Full system upgrade
sudo apt full-upgrade

# Check for updates
apt list --upgradable

# Dist-upgrade (major version update)
sudo apt dist-upgrade
```

### **CentOS/RHEL**

```bash
# Check for updates
sudo dnf check-update

# Install updates
sudo dnf upgrade

# Security updates only
sudo dnf upgrade --security

# Bugfix updates only
sudo dnf upgrade --bugfix
```

## **Part 7: Working with Repositories**

### **Ubuntu/Debian Repositories**

```bash
# View repositories
cat /etc/apt/sources.list

# View additional repositories
ls /etc/apt/sources.list.d/

# Add repository (example: PPA)
sudo add-apt-repository ppa:user/ppa-name

# Add generic repository
sudo add-apt-repository "deb http://archive.ubuntu.com/ubuntu focal main"

# Remove repository
sudo add-apt-repository --remove ppa:user/ppa-name

# Update after adding repository
sudo apt update
```

### **CentOS/RHEL Repositories**

```bash
# View enabled repositories
sudo dnf repolist

# View all repositories
sudo dnf repolist all

# Disable repository
sudo dnf config-manager --set-disabled repo-name

# Enable repository
sudo dnf config-manager --set-enabled repo-name

# Add repository
sudo dnf config-manager --add-repo http://repo-url

# View repository configuration
ls /etc/yum.repos.d/
cat /etc/yum.repos.d/repo-name.repo
```

## **Part 8: Practice Exercises**

### **Exercise 1: Basic Package Operations (Ubuntu)**

```bash
# 1. Update package lists
sudo apt update

# 2. Search for a package
apt search curl

# 3. Install package
sudo apt install curl

# 4. Verify installation
which curl
curl --version

# 5. Show package information
apt info curl
```

```bash

which wget

apt list --installed | wc -l
```

```bash

apt search python3

# 3. Install package (handles dependencies)
```

## **Part 10: Troubleshooting Package Issues**

### **Fix Broken Dependencies**

```bash
# Fix broken install
sudo apt --fix-broken install

# Fix missing packages
sudo apt install -f

# Clean and update
sudo apt clean
sudo apt update

# Try install again
sudo apt install package-name
```

### **Handle Locked Packages**

```bash
# Check for lock files
sudo lsof /var/lib/apt/lists/lock

# Wait for background processes to finish
sudo apt autoremove
sudo apt autoclean

# Force package configuration
sudo dpkg --configure -a

# Try again
sudo apt update
```
