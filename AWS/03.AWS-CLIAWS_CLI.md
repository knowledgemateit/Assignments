## **Part 1: AWS CLI Installation**

### **On Windows**

```powershell
# Option 1: Using MSI installer
# Download from: https://aws.amazon.com/cli/
# Run installer and follow wizard

# Option 2: Using Chocolatey
choco install awscli -y

# Option 3: Using Python pip
pip install awscliv2

# Verify installation
aws --version
```

### **On Linux/Mac**

```bash
# Using curl
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install

# Using brew (Mac)
brew install awscli

# Using apt (Ubuntu/Debian)
sudo apt update && sudo apt install -y awscli

# Verify installation
aws --version
```

## **Part 2: AWS CLI Configuration**

### **First-Time Configuration**

```bash
# Interactive setup
aws configure

# Prompts for:
# AWS Access Key ID: [from AWS Console]
# AWS Secret Access Key: [from AWS Console]
# Default region: us-east-1
# Default output format: json (or yaml, table, text)
```

### **Get Access Keys**

1. In AWS Console, go to **IAM → Users → Your Username**
2. Go to **Security credentials**
3. Create access key (if needed)
4. Copy **Access Key ID** and **Secret Access Key**
5. Store securely (don't share!)

### **Create Named Profiles**

```bash
# Create profile for different accounts/regions
aws configure --profile dev

# Create profile for production
aws configure --profile prod

# Use specific profile
aws ec2 describe-instances --profile prod
```

### **View Configuration**

```bash
# View default configuration
cat ~/.aws/credentials
cat ~/.aws/config

# List configured profiles
aws configure list

# List with specific profile
aws configure list --profile prod
```

## **Part 3: EC2 Management via CLI**

### **Describe Instances**

```bash
# List all instances
aws ec2 describe-instances

# List in JSON format (default)
aws ec2 describe-instances --output json

# List in table format
aws ec2 describe-instances --output table

# List in text format
aws ec2 describe-instances --output text

# Get specific information (query)
aws ec2 describe-instances --query 'Reservations[*].Instances[*].[InstanceId,InstanceType,State.Name]'

# Filter running instances
aws ec2 describe-instances --filters "Name=instance-state-name,Values=running"

# Filter by tag
aws ec2 describe-instances --filters "Name=tag:Name,Values=my-instance"
```

### **Launch Instance**

```bash
# Simple launch (uses default VPC, subnet, security group)
aws ec2 run-instances \
  --image-id ami-0c55b159cbfafe1f0 \
  --count 1 \
  --instance-type t3.micro \
  --key-name my-key-pair \
  --security-groups my-security-group

# With tags
aws ec2 run-instances \
  --image-id ami-0c55b159cbfafe1f0 \
  --instance-type t3.micro \
  --key-name my-key-pair \
  --tag-specifications 'ResourceType=instance,Tags=[{Key=Name,Value=cli-instance}]'

# Capture instance ID
INSTANCE_ID=$(aws ec2 run-instances ... --query 'Instances[0].InstanceId' --output text)
```

### **Stop and Start Instances**

```bash
# Stop instance
aws ec2 stop-instances --instance-ids i-1234567890abcdef0

# Start instance
aws ec2 start-instances --instance-ids i-1234567890abcdef0

# Reboot instance
aws ec2 reboot-instances --instance-ids i-1234567890abcdef0

# Terminate instance
aws ec2 terminate-instances --instance-ids i-1234567890abcdef0
```

### **Manage Security Groups**

```bash
# List security groups
aws ec2 describe-security-groups

# Create security group
aws ec2 create-security-group \
  --group-name cli-sg \
  --description "Security group created via CLI"

# Get security group ID
SG_ID=$(aws ec2 describe-security-groups --filters "Name=group-name,Values=cli-sg" --query 'SecurityGroups[0].GroupId' --output text)

# Add ingress rule (SSH)
aws ec2 authorize-security-group-ingress \
  --group-id $SG_ID \
  --protocol tcp \
  --port 22 \
  --cidr 0.0.0.0/0

# Add ingress rule (HTTP)
aws ec2 authorize-security-group-ingress \
  --group-id $SG_ID \
  --protocol tcp \
  --port 80 \
  --cidr 0.0.0.0/0

# Remove ingress rule
aws ec2 revoke-security-group-ingress \
  --group-id $SG_ID \
  --protocol tcp \
  --port 22 \
  --cidr 0.0.0.0/0

# Delete security group
aws ec2 delete-security-group --group-id $SG_ID
```

### **Manage Key Pairs**

```bash
# List key pairs
aws ec2 describe-key-pairs

# Create key pair
aws ec2 create-key-pair --key-name my-new-key --query 'KeyMaterial' --output text > my-new-key.pem

# Set permissions
chmod 400 my-new-key.pem

# Delete key pair
aws ec2 delete-key-pair --key-name my-new-key
```

## **Part 4: EBS and Snapshots via CLI**

### **Manage Volumes**

```bash
# List volumes
aws ec2 describe-volumes

# Create volume
aws ec2 create-volume \
  --size 20 \
  --volume-type gp3 \
  --availability-zone us-east-1a \
  --tag-specifications 'ResourceType=volume,Tags=[{Key=Name,Value=cli-volume}]'

# Describe specific volume
aws ec2 describe-volumes --volume-ids vol-1234567890abcdef0

# Delete volume
aws ec2 delete-volume --volume-id vol-1234567890abcdef0
```

### **Manage Snapshots**

```bash
# List snapshots
aws ec2 describe-snapshots --owner-ids self

# Create snapshot
aws ec2 create-snapshot \
  --volume-id vol-1234567890abcdef0 \
  --description "Snapshot via CLI"

# Monitor snapshot
aws ec2 describe-snapshots --snapshot-ids snap-1234567890abcdef0

# Copy snapshot to another region
aws ec2 copy-snapshot \
  --source-region us-east-1 \
  --source-snapshot-id snap-1234567890abcdef0 \
  --destination-region eu-west-1

# Delete snapshot
aws ec2 delete-snapshot --snapshot-id snap-1234567890abcdef0
```

## **Part 5: S3 Management via CLI**

### **Bucket Operations**

```bash
# List buckets
aws s3 ls

# Create bucket
aws s3 mb s3://my-unique-bucket-name

# Delete empty bucket
aws s3 rb s3://my-unique-bucket-name

# List bucket contents
aws s3 ls s3://my-bucket/

# List with details
aws s3 ls s3://my-bucket/ --recursive --human-readable --summarize
```

### **File Operations**

```bash
# Upload file
aws s3 cp file.txt s3://my-bucket/

# Download file
aws s3 cp s3://my-bucket/file.txt ./

# Sync directory (upload)
aws s3 sync ./local-dir/ s3://my-bucket/remote-dir/

# Sync directory (download)
aws s3 sync s3://my-bucket/remote-dir/ ./local-dir/

# Sync with delete
aws s3 sync s3://my-bucket/ ./local-dir/ --delete

# Copy within S3
aws s3 cp s3://bucket1/file.txt s3://bucket2/file.txt
```

## **Part 6: Automation Scripts**

### **Script 1: Launch and Configure Instance**

```bash
#!/bin/bash

# Variables
KEY_NAME="cli-key"
SG_NAME="cli-sg"
AMI_ID="ami-0c55b159cbfafe1f0"  # Ubuntu 22.04 in us-east-1
INSTANCE_TYPE="t3.micro"

echo "Creating security group..."
aws ec2 create-security-group \
  --group-name $SG_NAME \
  --description "Created by script"

SG_ID=$(aws ec2 describe-security-groups --filters "Name=group-name,Values=$SG_NAME" --query 'SecurityGroups[0].GroupId' --output text)

echo "Adding security group rules..."
aws ec2 authorize-security-group-ingress \
  --group-id $SG_ID \
  --protocol tcp \
  --port 22 \
  --cidr 0.0.0.0/0

aws ec2 authorize-security-group-ingress \
  --group-id $SG_ID \
  --protocol tcp \
  --port 80 \
  --cidr 0.0.0.0/0

echo "Launching instance..."
aws ec2 run-instances \
  --image-id $AMI_ID \
  --instance-type $INSTANCE_TYPE \
  --key-name $KEY_NAME \
  --security-group-ids $SG_ID \
  --tag-specifications 'ResourceType=instance,Tags=[{Key=Name,Value=cli-automated}]'

echo "Done!"
```

### **Script 2: Backup All Volumes**

```bash
#!/bin/bash

echo "Creating snapshots of all volumes..."

aws ec2 describe-volumes --query 'Volumes[*].VolumeId' --output text | \
while read volume_id; do
  echo "Creating snapshot for volume: $volume_id"
  
  aws ec2 create-snapshot \
    --volume-id $volume_id \
    --description "Automated backup on $(date)"
done

echo "All snapshots created!"
```

### **Script 3: Monitor Instances**

```bash
#!/bin/bash

echo "=== Running Instances ==="
aws ec2 describe-instances \
  --filters "Name=instance-state-name,Values=running" \
  --query 'Reservations[*].Instances[*].[InstanceId,InstanceType,PublicIpAddress,Tags[?Key==`Name`].Value|[0]]' \
  --output table

echo ""
echo "=== Stopped Instances ==="
aws ec2 describe-instances \
  --filters "Name=instance-state-name,Values=stopped" \
  --query 'Reservations[*].Instances[*].[InstanceId,InstanceType]' \
  --output table
```
