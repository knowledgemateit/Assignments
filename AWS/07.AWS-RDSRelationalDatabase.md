## **Part 2: Create RDS Instance**

```bash
# Via Console: RDS → Create database
# Configuration:
# - Engine: MySQL 8.0
# - Instance class: db.t3.micro (Free Tier)
# - Storage: 20 GB
# - Master username: admin
# - Password: (secure password)
# - VPC: Default
# - Public accessibility: Yes (for testing)

# Via CLI:
aws rds create-db-instance \
  --db-instance-identifier mydb \
  --engine mysql \
  --db-instance-class db.t3.micro \
  --master-username admin \
  --master-user-password MyPassword123! \
  --allocated-storage 20
```

## **Part 3: Connect to Database**

```bash
# Get endpoint
aws rds describe-db-instances --query 'DBInstances[0].Endpoint.Address' --output text

# Connect via MySQL client
mysql -h mydb.xxxx.us-east-1.rds.amazonaws.com -u admin -p

# Create database
CREATE DATABASE myapp;

# Create table
USE myapp;
CREATE TABLE users (
  id INT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(100),
  email VARCHAR(100)
);
```

## **Part 4: Backup and Snapshots**

```bash
# Automatic backups: Enabled by default (7 days retention)

# Create manual snapshot
aws rds create-db-snapshot \
  --db-instance-identifier mydb \
  --db-snapshot-identifier mydb-snapshot-2024-01

# List snapshots
aws rds describe-db-snapshots

# Restore from snapshot
aws rds restore-db-instance-from-db-snapshot \
  --db-instance-identifier mydb-restored \
  --db-snapshot-identifier mydb-snapshot-2024-01
```

---

## **Part 5: Multi-AZ and Read Replicas**

```bash
# Enable Multi-AZ
aws rds modify-db-instance \
  --db-instance-identifier mydb \
  --multi-az \
  --apply-immediately

# Create read replica
aws rds create-db-instance-read-replica \
  --db-instance-identifier mydb-read-replica \
  --source-db-instance-identifier mydb

# Multi-region read replica
aws rds create-db-instance-read-replica \
  --db-instance-identifier mydb-read-eu \
  --source-db-instance-identifier mydb \
  --source-region us-east-1 \
  --region eu-west-1
```

## **Part 6: Monitoring**

```bash
# View metrics in CloudWatch
# CPU, Memory, Disk I/O, Connections

# Enhanced monitoring
aws rds modify-db-instance \
  --db-instance-identifier mydb \
  --enable-cloudwatch-logs-exports error general slowquery \
  --apply-immediately

# View logs
aws logs tail /aws/rds/instance/mydb/error --follow
```

## **Part 7: Security**

```bash
# Use VPC security groups
# Restrict database port (3306 for MySQL) to application only

# Enable encryption
aws rds create-db-instance \
  --db-instance-identifier mydb \
  --engine mysql \
  --storage-encrypted

# Enable automated backups
aws rds modify-db-instance \
  --db-instance-identifier mydb \
  --backup-retention-period 30 \
  --apply-immediately
```
