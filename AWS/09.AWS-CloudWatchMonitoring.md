## **Part 1: CloudWatch Fundamentals**

CloudWatch provides:
- Metrics (performance data)
- Logs (application/system logs)
- Alarms (threshold monitoring)
- Dashboards (visualization)
- Events (automated actions)

---

## **Part 2: Metrics and Monitoring**

```bash
# View EC2 metrics
aws cloudwatch list-metrics --namespace AWS/EC2

# Get CPU metrics
aws cloudwatch get-metric-statistics \
  --namespace AWS/EC2 \
  --metric-name CPUUtilization \
  --dimensions Name=InstanceId,Value=i-1234567890abcdef0 \
  --start-time 2024-01-01T00:00:00Z \
  --end-time 2024-01-02T00:00:00Z \
  --period 300 \
  --statistics Average,Maximum
```

---

## **Part 3: Alarms**

```bash
# Create alarm for high CPU
aws cloudwatch put-metric-alarm \
  --alarm-name high-cpu \
  --alarm-description "Alert when CPU > 80%" \
  --metric-name CPUUtilization \
  --namespace AWS/EC2 \
  --statistic Average \
  --period 300 \
  --threshold 80 \
  --comparison-operator GreaterThanThreshold \
  --evaluation-periods 2 \
  --alarm-actions arn:aws:sns:us-east-1:123456789012:MyTopic

# Disable alarm
aws cloudwatch disable-alarm-actions --alarm-names high-cpu

# Delete alarm
aws cloudwatch delete-alarms --alarm-names high-cpu
```

## **Part 4: Logs**

```bash
# Create log group
aws logs create-log-group --log-group-name /aws/application

# Create log stream
aws logs create-log-stream \
  --log-group-name /aws/application \
  --log-stream-name instance-1

# Put log events
aws logs put-log-events \
  --log-group-name /aws/application \
  --log-stream-name instance-1 \
  --log-events timestamp=$(date +%s000),message="Application started"

# View logs
aws logs get-log-events \
  --log-group-name /aws/application \
  --log-stream-name instance-1

# Filter logs
aws logs filter-log-events \
  --log-group-name /aws/application \
  --filter-pattern "ERROR"
```

## **Part 5: Dashboards**

```bash
# Create dashboard
aws cloudwatch put-dashboard \
  --dashboard-name my-dashboard \
  --dashboard-body '{
    "widgets": [{
      "type": "metric",
      "properties": {
        "metrics": [
          ["AWS/EC2", "CPUUtilization", {"stat": "Average"}],
          ["AWS/RDS", "DatabaseConnections"]
        ],
        "period": 300,
        "stat": "Average",
        "region": "us-east-1"
      }
    }]
  }'

# List dashboards
aws cloudwatch list-dashboards

# Delete dashboard
aws cloudwatch delete-dashboards --dashboard-names my-dashboard
```

## **Part 6: SNS Notifications**

```bash
# Create SNS topic
aws sns create-topic --name my-alerts

# Get topic ARN
TOPIC_ARN=$(aws sns list-topics --query 'Topics[0].TopicArn' --output text)

# Subscribe to topic
aws sns subscribe \
  --topic-arn $TOPIC_ARN \
  --protocol email \
  --notification-endpoint john@example.com

# Publish message
aws sns publish \
  --topic-arn $TOPIC_ARN \
  --message "Alert: High CPU detected"
```
